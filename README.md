# cookiedclaw landing

Static single-page landing for [cookiedclaw](https://github.com/cookiedclaw/cookiedclaw) — the Telegram-resident agent that lives inside Claude Code.

## Local

```sh
python3 -m http.server 4747
# open http://localhost:4747
```

## Deploy

Deployed via Vercel. Pure static HTML/CSS/JS — no build step.
