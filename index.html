04how it feels

Your VPS, texting you back.

Run Claude Code on a server, a home box, a forgotten mac mini — then drive it from the train, the kitchen, anywhere a Telegram thread reaches.

The agent is resident in the chat. Not polled, not relayed — there.

where the official telegram@claude-plugins-official plugin gives you a thread, cookiedclaw gives you a workspace — identity files, a setup wizard, an auto-mirrored slash menu, and live tool progress streaming back as it runs.

05ninety seconds

If you have Claude Code, you have this.

~/$ cookiedclaw setup
$ claude plugin marketplace add cookiedclaw/cookiedclaw
$ claude plugin install cookiedclaw@cookiedclaw
$ mkdir -p ~/cookiedclaw && cd ~/cookiedclaw
$ claude --dangerously-load-development-channels plugin:cookiedclaw@cookiedclaw
> /cookiedclaw:setup
wizard runs · botfather token · you'll be texting your repo in ~90 seconds
or read the source — it's small.
